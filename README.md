# Portfolio Project

This is a modern, responsive portfolio website built with React, Vite, and Tailwind CSS.

## Features

- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Modern Tech Stack**: React, Vite, TypeScript, and Tailwind CSS.
- **Interactive UI**: Smooth animations and transitions using Framer Motion and Lucide Icons.
- **Customizable**: Easy to update with your own content and styling.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   # OR if you use bun
   bun install
   ```

3. Start the development server:
   ```sh
   npm run dev
   # OR if you use bun
   bun run dev
   ```
